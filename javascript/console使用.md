# console使用

```javascript
// 参考地址: https://mp.weixin.qq.com/s/IeosI8ty3e9vDl7wJyZu5g
console.time()
console.timeEnd()

// 日志位置信息
console.trace()

// 堆栈信息 - 属性
console.memory

// 打印表格
console.table()

// 条件日志
console.assert(condition, msg)

// 次数
console.count()

// 启动和结束浏览器性能工具 - “performance profile”,帮助您精确地分析您想要的内容，并避免了您单击鼠标，它取决于程序执行时间。
console.profile(‘profileName’)
console.profileEnd('profileName')

console.group()
console.groupEnd()

```
